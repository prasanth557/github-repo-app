"# github-repo-app" 

## Installation

1. Clone the repository:

   ```bash
   git clone <repository-url>

2. Navigate to the project folder:

   cd github-repo-app

3. Install dependencies:
   
   npm install
   npm i @ng-bootstrap/ng-bootstrap
   npm install rxjs
   
4. Start the development server:

   ng serve
